A github template to build dynamic webapps in HTML, Bulma CSS and Go.

February 11, 2021    

Sometimes I find myself in need of a simple prototyping tool to visualise an user interface and find myself deep in figma. Figma(and others like it) can be simple enough to start with, but a good looking while easy to build interface seems always further away.

Since I have always tried to be good enough with the languages of the web: HTML & CSS and I come across using Go quite regularly due to my primary job as a devops engineer, I wondered whether I can give myself a head start in using these skill sets to prototype user interfaces quickly.

gomodest-template is my attempt to do so.
